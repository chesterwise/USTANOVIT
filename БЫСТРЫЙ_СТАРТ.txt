📋 ЧТО НУЖНО ПОДГОТОВИТЬ:

1. ✅ Сервер Ubuntu 22.04 на Timeweb
2. ✅ SSH доступ с правами root
3. ✅ Telegram Bot Token (получить у @BotFather)
4. ✅ OpenAI API Key (опционально)


🎯 УСТАНОВКА В 3 ШАГА:

════════════════════════════════════════════════════════════════

ШАГ 1: Подключитесь к серверу
────────────────────────────────────────────────────────────────
ssh root@ваш_ip_адрес


ШАГ 2: Скачайте и запустите проект
────────────────────────────────────────────────────────────────
cd /root
git clone https://github.com/your-username/finance-bot.git
cd finance-bot
chmod +x *.sh
./install-timeweb.sh


ШАГ 3: Настройте Telegram webhook
────────────────────────────────────────────────────────────────
./setup-telegram-webhook.sh

════════════════════════════════════════════════════════════════


✨ ВСЁ ГОТОВО! Отправьте сообщение боту в Telegram!


📊 УПРАВЛЕНИЕ БОТОМ:
────────────────────────────────────────────────────────────────

# Интерактивное меню управления
./manage-bot.sh

# Или используйте команды напрямую:
pm2 list                    # Статус всех процессов
pm2 logs                    # Просмотр логов
pm2 restart all             # Перезапуск бота


🔗 ПОЛУЧИТЬ URL CLOUDFLARE TUNNEL:
────────────────────────────────────────────────────────────────
journalctl -u cloudflare-tunnel | grep trycloudflare.com


🔄 ОБНОВЛЕНИЕ БОТА:
────────────────────────────────────────────────────────────────
cd /root/finance-bot
git pull
npm install
npm run build
pm2 restart all
./setup-telegram-webhook.sh


🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:
────────────────────────────────────────────────────────────────

1. Проверьте логи:
   pm2 logs finance-bot

2. Проверьте статус служб:
   pm2 list
   systemctl status cloudflare-tunnel

3. Проверьте webhook:
   curl "https://api.telegram.org/bot<ТОКЕН>/getWebhookInfo"

4. Перезапустите всё:
   pm2 restart all
   systemctl restart cloudflare-tunnel
